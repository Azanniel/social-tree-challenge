## 💻 Projeto

O Social Tree Challenge foi um desafio proposto na plataforma discovery da Rocketseat 💜 para masterizar o aprendizado de HTML5 e CSS3.

Neste desafio pude aprender muito sobre CSS e garantir que meus designs sejam elegantes e responsivos para que todos possam ver em qualquer dispositvos.
Além disso pude criar minha própria social tree para mandar para meus amigos e até adicionar nos aplicativos de redes sociais.

Você pode conferir [clicando aqui.](https://azanniel.github.io/social-tree-challenge/)
